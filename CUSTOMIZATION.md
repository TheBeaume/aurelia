# AURELIA Customization Guide

Thank you for choosing AURELIA.

This guide will help you personalize your website without changing the overall design.

---

# Change Salon Name

Open:

src/data/siteConfig.js

Update:

- Salon Name
- Tagline
- Phone Number
- Email Address
- WhatsApp Number
- Address
- Social Media Links

---

# Change Logo

Replace your logo file inside:

src/assets/

Update the logo reference in the Navbar component if required.

---

# Change Theme Color

Open:

src/App.jsx

Find:

--accent

Replace the color with your preferred brand color.

Example:

Gold
Black
Rose Gold
Purple
Emerald

---

# Replace Images

All website images are stored inside:

src/assets/images/

Simply replace the image while keeping the same filename.

Examples:

hero.png

team1.png

gallery1.png

before.png

after.png

---

# Edit Services

Open:

src/components/Services.jsx

Update:

- Service Name
- Description
- Image

---

# Edit Signature Packages

Open:

src/components/Pricing.jsx

Update:

- Package Name
- Features

---

# Edit Team

Open:

src/components/Team.jsx

Update:

- Name
- Role
- Photo

---

# Edit Gallery

Replace the images inside:

src/assets/images/

No additional code changes are required.

---

# Edit Contact Information

Open:

src/data/siteConfig.js

Update:

- Phone
- Email
- Address
- Google Maps
- Social Links

---

# SEO

index.html

- Website Title

- Keywords
- Open Graph Image

public/sitemap.xml

public/robots.txt
