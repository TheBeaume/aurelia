# Getting Started

1. Read **README.md**
2. Follow **INSTALLATION.md**
3. Personalize your website using **CUSTOMIZATION.md**
4. Build your project:

```bash
npm install
npm run dev
```

Create a production build:

```bash
npm run build
```
