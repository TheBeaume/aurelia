# AURELIA Installation Guide

Thank you for purchasing AURELIA.

## Requirements

- Node.js 20+
- npm

## Installation

Clone the project or extract the ZIP.

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Deployment

AURELIA can be deployed on:

- Vercel (Recommended)
- Netlify
- Any static hosting provider
