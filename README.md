# AURELIA

## Luxury Salon Website Template

AURELIA is a premium React + Vite + Tailwind CSS website template designed for luxury salons, spas, beauty studios and wellness brands.

## Features

- Premium Luxury Design
- Fully Responsive
- SEO Optimized
- React + Vite
- Tailwind CSS
- AI Generated Images
- About Page
- Privacy Policy
- Terms & Conditions
- Contact Section
- Theme Switcher
- Fast Loading
- Easy Customization

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion

## Installation

```bash
npm install
npm run dev
```

Production Build

```bash
npm run build
```

## Folder Structure

```
src/
components/
pages/
assets/
public/
```
